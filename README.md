约跑 - 用于 Android 8.0+ 的无需 ROOT 权限的虚拟定位 APP

## 简介
&emsp;&emsp;约跑是一个基于 Android 调试 API + 百度地图及定位 SDK 实现的安卓定位修改工具，并且同时实现了一个可以自由控制移动的摇杆。使用约跑，不需要 ROOT 权限就可以随意修改自己的当前位置以及模拟移动。

## 功能
1. 定位修改
2. 摇杆控制移动
3. 历史记录
4. 位置搜索
5. 直接输入坐标
6. 轨迹绘制
7. 循迹移动
8. 随即速度偏移
